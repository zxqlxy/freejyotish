FREEJYOTISH
Open-source Vedic Astrology Software

FreeJyotish is an open-source Vedic astrology application. It is written in the Java programming language, and is intended to run on any operating system that uses Java. FreeJyotish is still in the design and construction phase. If you are a Java programmer, graphic designer, font designer, internationalization expert (particularly Indian languages) or xml wizard who is interested in working on this project, please contact us.

Furthermore, the FreeJyotish project is looking for a computer donation. Any computer (PII or better or equivalent) would be welcome to help set up a webserver.

Some Cool Features - The First Release of Free Jyotish Is Planned to Include:

- Ultra-accurate "Swiss Ephemeris" back end. This program, from Astrodienst, is based on JPL planetary data files, and achieves accuracy of 1/100 arc/sec. The Swiss Ephemeris is the calculation engine used by most professional astrology programs sold today.
- All basic jyotish functions, such as: rasi chart, all varga charts, all astakavarga charts, shadbala, vimshottari, and more.
- Highly extensible, flexible core application, perfectly suited for many different platforms
- Professional look and feel
- Much more.
